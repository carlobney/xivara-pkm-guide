Don't Do This

  • Duplicate content ‚Äî Link or embed, never copy
  • Edit others' workspaces ‚Äî Ask the steward first
  • Put execution here ‚Äî Tasks go in Linear, code in GitHub
  • Move files without updating links ‚Äî Breaks the graph
  • Delete files ‚Äî Archive to Archive/YYYY/ instead
  • Link to unverified files ‚Äî Check it exists first

Daily Usage

  • Morning: Check [[Dashboard]] for status and focus
  • Quick capture: Drop notes in 00-Inbox/
  • Meetings: Use Templates/Meeting-Note
  • Find anything: Cmd+Shift+F (search) or Cmd+O (quick switch)
  • Before creating: Search first ‚Äî it might exist
